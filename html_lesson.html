First Steps

Getting started

Copy and paste this into your file. These are the fundamental building blocks of a web page: 


<!DOCTYPE html>

<html>

</html>
  • <!DOCTYPE html> indicates that this is an html document (rather than an image, or some other web file)
  • <html> indicates the beginning of the html code, and </html> indicates the end
Head

Copy and paste this between the <html> and </html>. This is the head, which contains metadata (invisible information) about your page 


	<head>
		<title>My website</title>
		<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">
	</head>
  • <head> and </head> indicate the start and end of the head
  • <title> and </title> enclose the title of the page. This is what you see in the tab at the top of the browser
  • <link ...> allows us to use some third party resources that automatically make spacing easier
Body

Copy and paste this after the </head> but before the </html>. This is the body, which contains all of the visible elements on the page 


	<body>
		Hello, world!
	</body>
  • <body> and </body> indicate the start and end of the body
  • "Hello, world!" is the first thing your program says, to make sure it's working :) We will remove this when we make the actual website

Elements

Divs

The first step to building your website is to divide everything into a grid of rectangles. My example website obviously has three rectangles from top to bottom, but it also has invisible rectangles holding the other pieces of content in place. First, to make the visible rectangles, I want something like this. Each div (division) is a colored box. For this example, erase "Hello, world!" and copy and paste this in:


		<div class="row" style="background-color: rgb(0,0,0); padding: 70px">
			First rectangle content will go here
		</div>
		<div class="row" style="background-color: rgb(22,132,198); padding: 70px">
			Second rectangle content will go here
		</div>
		<div class="row" style="background-color: rgb(83,255,167); padding: 70px">
			Third rectangle content will go here
		</div>
  • <div...> and </div> indicate the start and end of each div
  • class="row" uses that third party resource to make a nice horizontal rectangle that fills the whole width of the page
  • style="..." sets the style of the div
    • background-color is the background color of the rectangle. It uses RGB color. So you just need to say rgb(255,0,0) to set the color to red, rgb(0,255,0) to set the color to green, etc.
    • padding is the amount of extra space around the elements inside the rectangle. Right now there are no elements inside, so the rectangle will be 140 (two times 70) pixels tall. It uses the unit pixels, which means labeling with "px"

Then we need dividers to separate the space inside each visible grid. We do this by splitting the entire screen into six columns. Referring to the lines drawn on your paper, figure out how many columns each item needs to take up. For example, my avatar photo takes up about 5/12 columns, and my name takes up about 7/12 columns. Then we create invisible container divs to hold these pieces. Vertical divider divs go inside row divs. For this example, erase "First rectangle content will go here" and copy and paste this in:


			<div class="col-md-5">
				Avatar will go here
			</div>
			<div class="col-md-7">
				Name will go here
			</div>
  • class="col-md-5" means that the div will be 5/12 columns wide
  • class="col-md-7" means that the div will be 7/12 columns wide

I also have these dividers for the second rectangle (replace "Second rectangle content will go here"):


			<div class="col-md-4">
				Game will go here
			</div>
			<div class="col-md-8">
				Paragraph describing game will go here
			</div>

And these dividers for the third rectangle (replace "Third rectangle content will go here"):


			<div class="col-md-3">
			</div>
			<div class="col-md-6">
				YouTube video will go here
			</div>
			<div class="col-md-3">
			</div>
Text

Now that we have our dividers, let's put some stuff in there! The easiest place to start is text. I have a title containing my name, and a paragraph describing my game. For the title, I'm going to use an <h1> tag. h1 means header 1, which is the largest header. Then for the paragraph, I'm going to use an <h3> tag, AKA header 3, which is smaller than an <h1> or <h2> tag. You can also use <h4> and up, or <p> (paragraph) if you want much smaller text, like if you were writing an entire blog entry.

First, replace "Name will go here" with this:


				<h1 style="color: rgb(33,217,255); font-size: 100px">
					Erin
				</h1>
  • Whereas rectangles have their color set with background-color, text (headers and paragraphs) has its color set with just color. It uses rgb just the same as background-color
  • Since I want my name to be even more giant than <h1> is by default, I added a font-size attribute to make it 100px.

Second, replace "Paragraph describing game will go here" with this:


				<h3>
					This is a beautiful paragraph about my game.  My game features an alien space man character, a fence, and some coins.  When you get three coins, you win!  Then the game will restart.  It is very exciting.  There is a wall of grass/dirt blocks at the edge so the alien space man character does not go off the edge into the abyss.  At some point, I will likely add more features to this exciting game!
				</h3>
Embedded content

The awesome thing about modern web development is that we can embed content from other places directly into our websites! Today, I'm going to embed my browser game (from flowlab.io) and a video from YouTube

The first step to embedding something is getting the embed code. For your flowlab.io game, go log in to your flowlab.io profile, and you will notice underneath the Play and Edit buttons there is a box labeled "Embed". Copy everything in that box, and paste it in place of "Game will go here". My embed code looks like this:


				<iframe width="320" height="320" src="http://www.flowlab.io/game/embed/216628" frameborder="0" allowfullscreen></iframe>

Then go to YouTube, find a video you like, and scroll down so you can see the "Share" option immediately below the video. Click on "Share" and some new options should appear below. Click on "Embed". This will also give you a box of code. Copy everything in that box, and paste it in place of "YouTube video will go here". My embed code looks like this:


				<iframe width="560" height="315" src="https://www.youtube.com/embed/StTqXEQ2l-Y" frameborder="0" allowfullscreen></iframe>
  • src means the source of the embedded window. So, a flowlab.io game has a source on the flowlab.io website, and a YouTube video has a source on the YouTube website
Images

Last but not least, images are a fun addition to your website. For today, let's get our "profile" image from this set of cartoon octocats. What's an octocat, you ask? A hybrid octopus cat. It's adorable. It's also the mascot of GitHub, the internet service I'll be using to publish your websites on the web. Click this link to pick one out. Once you have one you like, right click on it and select "Open Image in New Tab". The URL that appears in the browser top bar will be the source file for your image. My image code looks like this (copy and paste this in place of "Avatar will go here"):


				<img src="https://octodex.github.com/images/gracehoppertocat.jpg" style="height: 300px; border-radius: 50%"></img>
  • src means the source of the image, same as how embedded windows have a source
  • I set the height to 300px to get it just how I want it, but if you leave this out, the image will be the original size from the GitHub website.
  • border-radius is what turns this square image into a circle. Try out some different percentages and see what they do!
